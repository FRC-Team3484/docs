# Maintaining Open Scouting

[Open Scouting](https://github.com/FRC-Team3484/open-scouting) is 3484's custom scouting app. A few steps are required to prepare the application for the new season. You will need to be an admin on the Open Scouting server to perform these steps.

1. Create a new season on the admin page and mark it as active.
2. Go to "Manage match scouting fields" and ensure you select your new season. Add game pieces for this season's game
3. Then, create match scouting fields for the information that you want to collect.
4. Go back and click "Manage pit scouting questions". Ensure you select your new season, and create questions based on what you want scouts to ask during pit scouting.

Open Scouting is now ready for a new season! Clients will need to rebuild their event and season data cache to see your changes.
